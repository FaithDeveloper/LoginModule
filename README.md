# LoginModule
This is Login Model for Swift3

If you want to use LoginModel, 
1. Install Pod.
2. Firebase connect

### 1. Pods for LoginModel.
  * pod 'Firebase/Core'
  * pod 'Firebase/Auth'
  * pod 'GoogleSignIn'
  * pod 'Firebase/Database'
  
### 2. How to Connect Firebase.
